# flashcard_quiz
A Flutter-based flashcard quiz app designed for effective learning. Users can create, edit, and delete flashcards, flip cards to reveal answers, track learning progress, and switch between light and dark modes. Data is stored locally using SharedPreferences, making the app fast, offline-friendly, and easy to use.
